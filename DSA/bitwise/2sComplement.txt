2's complement:
-------------------------------------------- [
n + ~n + 1 = 0
say n=5
5 + ~5 + 1 =0
00....0101 	-> 5
11....1010	-> ~5
00....0001	-> 1
------------
00....0000	-> sum is 0, int overflow, carry 1
so, 1's complement is just flipping the bits
~n = -n -1
~5 = -5-1 = -6
00....0101 	-> 5
11....1010	-> ~5 (= -6) -> 1's complement
00....0001	-> +1
----------
11....1011	-> -5	-> 2's complement ( = 1's complement + 1)
-------------------------------------------- ]
